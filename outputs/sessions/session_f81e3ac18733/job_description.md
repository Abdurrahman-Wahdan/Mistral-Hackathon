# Job Description: AI Engineer

**Department:** AI Platform / Product Engineering  
**Level:** Mid/Senior  
**Employment Type:** Full-time

## Role Summary
Build and operate AI-powered product features end-to-end. You will design LLM workflows, retrieval pipelines, and evaluation loops that are reliable, cost-aware, and production ready.

## Core Responsibilities
- Build agentic and retrieval-based workflows for real user problems.
- Design prompts, structured outputs, and tool-calling flows with guardrails.
- Implement evaluation pipelines for quality, safety, and regression detection.
- Optimize latency/cost while preserving answer quality.
- Collaborate with product and backend teams to ship AI features safely.

## Required Tech Stack
- **Languages:** Python, TypeScript (preferred)
- **LLM Tooling:** LangChain/LangGraph or equivalent orchestration stack
- **APIs:** OpenAI/Mistral/Anthropic API integration patterns
- **RAG:** Embeddings, reranking, vector databases (Qdrant/Pinecone/Weaviate)
- **Backend:** FastAPI, async I/O, queue/background processing
- **Observability:** Tracing, prompt/version tracking, evaluation dashboards

## What Good Looks Like
- Strong software engineering fundamentals beyond prompting.
- Ability to debug AI failures systematically using metrics and traces.
- Pragmatic architecture decisions with clear tradeoff reasoning.
