# Job Description: UX Designer

**Department:** Design  
**Level:** Mid/Senior  
**Employment Type:** Full-time

## Role Summary
Design intuitive, high-impact user experiences across web and mobile workflows. You will combine user research, interaction design, and visual craft to improve conversion and usability.

## Core Responsibilities
- Own UX process from discovery to high-fidelity design.
- Create wireframes, user flows, and interactive prototypes.
- Run usability testing and iterate based on evidence.
- Collaborate closely with PM and Engineering for implementation quality.
- Contribute to and evolve the design system.

## Required Skills & Tooling
- **Design Tools:** Figma, prototyping tools
- **Research:** Usability testing, surveys, interview synthesis
- **UX Craft:** Information architecture, interaction design, accessibility
- **Systems:** Design tokens, component libraries
- **Collaboration:** Handoff documentation and async feedback workflows

## What Good Looks Like
- Strong portfolio with measurable product impact.
- Deep user empathy with pragmatic execution.
- Consistent attention to interaction details and accessibility.
