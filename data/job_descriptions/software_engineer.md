# Job Description: Software Engineer

**Department:** Engineering  
**Level:** Mid/Senior  
**Employment Type:** Full-time

## Role Summary
Build and scale product features end-to-end, from API design and backend services to frontend integration. You will work closely with Product, Design, and Data to ship reliable software quickly.

## Core Responsibilities
- Design, implement, and maintain backend and frontend services.
- Write clean, testable, and production-grade code.
- Improve system performance, reliability, and observability.
- Collaborate on architecture decisions and technical roadmaps.
- Participate in code reviews and mentor junior engineers.

## Required Tech Stack
- **Languages:** TypeScript/JavaScript, Python
- **Frontend:** React, Next.js
- **Backend:** FastAPI or Node.js, REST APIs
- **Data:** PostgreSQL, Redis
- **Infra:** Docker, CI/CD, cloud deployment (AWS/GCP)
- **Practices:** Testing, monitoring, incident ownership

## What Good Looks Like
- Strong debugging and problem-solving under ambiguity.
- Ability to balance shipping speed with code quality.
- Clear communication across technical and non-technical teams.
