# Job Description: DevOps Engineer

**Department:** Platform / Infrastructure  
**Level:** Mid/Senior  
**Employment Type:** Full-time

## Role Summary
Build and operate secure, scalable cloud infrastructure and deployment pipelines. You will reduce operational risk, improve reliability, and accelerate engineering delivery.

## Core Responsibilities
- Build and maintain CI/CD pipelines for multiple services.
- Manage cloud infrastructure as code.
- Improve uptime through monitoring, alerting, and incident response.
- Optimize performance and cloud cost.
- Partner with security and engineering on hardening and compliance.

## Required Tech Stack
- **Cloud:** AWS/GCP
- **Containers:** Docker, Kubernetes
- **IaC:** Terraform
- **CI/CD:** GitHub Actions/GitLab CI
- **Observability:** Prometheus, Grafana, ELK/OpenTelemetry
- **Security:** Secrets management, IAM best practices

## What Good Looks Like
- Strong automation mindset and operational excellence.
- Fast incident triage with clear postmortem ownership.
- Ability to balance developer velocity with platform safety.
